# 동국제강 브랜드 가이드라인

## 색상 체계

### 주요 색상 (Primary)

| 이름 | 코드 | 용도 |

| **다크** | `#262626` | 본문 텍스트 |
| **화이트** | `#FFFFFF` | 배경 |

| **연청록** | `#B7D0D4` | 배경 강조 |

## 타이포그래피

| 용도 | 폰트 | 대체 |
|------|------|------|
| **제목** | 본고딕 Bold/Medium | Noto Sans KR Bold |
| **본문** | 본고딕 Normal | Noto Sans KR Regular |

### 크기

- 대제목: 32pt, 네이비
- Main Title: 19pt, 네이비
- Action Title: 17pt
- 본문: 16pt, 다크
- 캡션: 14pt
- 슬라이드 번호: 8pt

## 적용 패턴

### PPT

```
표지: 상단 그라데이션 (#C51F2A → #002452)
글머리: ▐ (L1), • (L2), – (L3)
```

### HTML/CSS

```css
:root {
  --dk-navy: #002452;
  --dk-red: #C51F2A;
  --dk-dark: #262626;
  --dk-gray: #757575;
  --dk-blue-gray: #4B6580;
  --dk-light-cyan: #B7D0D4;
}
h1, h2, h3 { color: var(--dk-navy); }
.accent { color: var(--dk-red); }
```

### 차트 팔레트

```python
DONGKUK_PALETTE = ['#002452', '#C51F2A', '#4B6580', '#B7D0D4', '#E9B86E', '#D55633']
```

## 디자인 요소

- **구분선**: #B6B6B6 (65%), 0.5pt
- **그라데이션**: 좌→우, #C51F2A → #002452
