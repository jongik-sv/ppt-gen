# 6단계 프로세스 그리드 템플릿

## 개요
이 템플릿은 `ppt-sample/깔끔이-딥그린.pptx` 파일의 슬라이드 5에서 추출한 6단계 프로세스 흐름을 표현하는 그리드 레이아웃입니다.

## 구조
- **그리드 레이아웃**: 3열 x 2행 (3 columns x 2 rows)
- **카드 개수**: 6개 (고정)
- **카드 구성**:
  - 번호 원 (Number Circle): 상단 좌측에 위치, 각 단계의 번호 (1-6)
  - 삼각형 장식 (Triangle Accent): 상단 우측에 위치, 시각적 강조
  - 둥근 카드 배경 (Rounded Rectangle): 전체 배경
  - 제목 (Title): 중앙에 위치, 단계 제목
  - 설명 (Description): 제목 아래, 상세 설명

## 색상 팔레트
| 요소 | 색상 | HEX |
|------|------|-----|
| 번호 원 & 삼각형 | 테마 보조색 (Teal/Green) | #A1BFB4 |
| 카드 배경 | 라이트 그레이 | #F5F7F6 |
| 제목 텍스트 | 다크 텍스트 | #1A1A1A |
| 설명 텍스트 | 미디엄 그레이 | #666666 |

## 타이포그래피
| 요소 | 크기 | 무게 |
|------|------|------|
| 제목 (Title) | 16pt | 600 (Semi-bold) |
| 설명 (Description) | 12pt | 400 (Regular) |
| 번호 (Number) | 32pt | 700 (Bold) |

## 간격 설정
- **카드 간 수평 간격**: 3%
- **카드 간 수직 간격**: 25%
- **카드 내부 패딩**: 20px
- **타이틀 상단 패딩**: 40px
- **번호 원 오버랩**: -20px (카드 상단 외부로 튀어나옴)

## 파일 구성
1. **template.yaml**: 템플릿 정의 및 설정
2. **template.html**: 기본 템플릿 HTML (기본 샘플 데이터)
3. **example.html**: 예제 HTML (소프트웨어 개발 생명주기 예제)
4. **thumbnail.png**: 템플릿 썸네일 (320x180)

## 사용 방법

### HTML 기반 사용
HTML 파일을 직접 수정하여 데이터를 변경합니다:

```html
<div class="card">
    <div class="number-circle">1</div>
    <div class="triangle-accent"></div>
    <div class="card-content">
        <div class="card-title">제목 입력</div>
        <div class="card-description">설명 입력</div>
    </div>
</div>
```

### JavaScript를 통한 데이터 주입
`templateData` 객체를 수정하여 동적으로 데이터를 삽입합니다:

```javascript
const templateData = {
    items: [
        {
            number: 1,
            title: "단계 1",
            description: "설명"
        },
        // ... 6개 항목
    ]
};
```
