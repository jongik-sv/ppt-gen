템플릿 설명

Percent Dot Grid는 비율 데이터를 시각적으로 표현하는 인포그래픽 템플릿입니다.

주요 특징

  • 10x10 도트 그리드 (총 100개 도트)
  • 하단 채우기 방식으로 퍼센트 시각화
  • 유연한 열 개수 (2~6 열 권장)
  • 각 항목별 제목 및 설명 텍스트 지원
  • 커스터마이징 가능한 색상 (도트, 텍스트)

플레이스홀더 구조

{
  "items": [
    {
      "percent": 56,
      "title": "항목 명",
      "description": "설명 텍스트",
      "dot_color": "#479374"
    }
  ]
}

스타일 커스터마이징

  • dot_color: 채워진 도트 색상 (기본: #479374)
  • Empty dot color: #E8E8E8 (CSS에서 변경)
  • Text color: #153325 (진하고 어두운 초록)
  • Percentage font size: 48px bold
  • Title font size: 18px semi-bold

사용 사례

  • 프로젝트 진행률 표시
  • 분야별 성과 달성도
  • 설문 조사 결과 시각화
  • 목표 달성도 비교
  • 역량 평가 점수 표시
  • 부서별 KPI 성과
